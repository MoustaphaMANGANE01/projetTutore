L'etat civil ,c'est quoi?

L'état civil est la situation de la personne dans la famille et la société,résultat d'une procédure écrite d'identification administrative.
L'état civil désigne également le service public chargé de dresser les actes constatant ces faits,notamment
les actes de naissance,de mariage et de décès.

Les différentes étapes pour obtenir un document d'etat sont ci-dessous :

Demande d’extrait de naissance

D'abord pour obtenir cette document vous devez vous enregistrer si vous n'avez pas encore de compte ou se connecter si vous avez déja un compte.Apres que la connexion sera faite,vous verrez une liste déroulante vous proposant les différents types de documents et vous choisissez: Le demande d'extrait de naissance.Aprés avoir fait votre choix vous cliquez sur le bouton "confirmer",puis un formulaire vous serez présenté et vous renseignez les informations suivantes:

• Nom.

• Prénom.

• Age.

• Numéro de téléphone.

• Numéro de registre et année d’enregistrement.

• Oubien votre numéro de carte d’identité nationale(si vous n'avez pas mémorisé votre nu,éro de registre).

Vous cliquez sur le bouton "Valider",aprés la validation ,une verification sera faite pour voir si vous votre document est bien présent dans les registres de l'état civil,si OUI vous verrez d'autres champs à remplir qui sont :

• Nombre de copies.

•Payer les droits de timbre par voie électronique(paypal,Orange Money ou autres ...

Apres avoir renseignez les champs ,vous cliquez sur le bouton "Terminer".Un message de Rendez-vous ,vous sera envoyé sur votre compte d'utilisateur.
